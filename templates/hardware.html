Test de Hardware

    * Herramientas de Monitoreo

    • HWMonitor (Gratuito): Muestra temperaturas, voltajes y uso de CPU/GPU en tiempo real.
    • HWiNFO (Gratuito): Más avanzado, con gráficos detallados y mucha información técnica.
    • Open Hardware Monitor (Gratuito): Alternativa de código abierto y ligera.

    * Test Tarjeta Gráfica

    • 3DMark (De pago, con prueba gratuita): Analiza el rendimiento de la tarjeta gráfica simulando juegos. (Steam)
    • Unigine Superposition (Gratuito): No probado.

    * Test CPU

    • Prime95 (Gratuito): Ideal para pruebas de estrés extremo y estabilidad del procesador.
    • Cinebench R23 (Gratuito): Evalúa el rendimiento del procesador.
    • Intel Processor Diagnostic Tool (Gratuito): Prueba completa para procesadores Intel.
    • Ryzen Master (Gratuito): Herramienta de monitoreo y ajuste para AMD Ryzen.

    * Test RAM

    • MemTest86 (Gratuito para uso básico, versión de pago disponible): Test exhaustivo de errores en la RAM.
    • Windows Memory Diagnostic (Gratuito): Herramienta integrada en Windows.

    * Test de SSD / HDD

    • CrystalDiskInfo (Gratuito): Muestra la salud del disco y su temperatura.
    • CrystalMarkRetro (Gratuito): Pruebas de rendimiento de disco, CPU, RAM y GPU.
    • HD Tune (De pago, con versión gratuita limitada): Diagnóstico de sectores defectuosos.

    * Test Motherboard y Fuente de Alimentación

    • HWMonitor (Gratuito): Para ver voltajes y temperaturas.
    • OCCT (De pago, con prueba gratuita): Prueba de estabilidad de placa base, CPU y fuente de alimentación.
